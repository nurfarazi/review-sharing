## Welcome to Review sharing platform

## Prerequisite

 - **M**ongoDB (4.*) 
 - **E**xpress (4.*) 
 - **A**ngular (8.0.*)
 - **N**ode.js (11.3.*)

###Install dependency
`npm install`

## Run the project
`concurrently -c "yellow.bold,green.bold" -n "SERVER,BUILD" "nodemon server" "ng build --watch"`
